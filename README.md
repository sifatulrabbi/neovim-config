# My Neovim configs

> Switched over to Neovim at November 20, 2023.

## Configs

- Basic keymaps.
- LSP with autoformatting.
- Common Git stuff.
- Color scheme.

## Requirements

Following dependencies are required for the config and the plugins to work.

- `ripgrep`
- `fd`
- `cmake` or `make`

```bash
# macos
brew install ripgrep fd cmake fzf

# arch
sudo pacman -S ripgrep fd xsel xclip fzf

# debian
sudo apt install ripgrep fd-find xsel xclip fzf
```
